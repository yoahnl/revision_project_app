# Documentation V1 — Questions riches fermées

Ce dossier isole la roadmap V1 de Revision App.

La V0 a validé le socle applicatif : documents, chunks, notions, résumé, fiche, QCM v3, question ouverte corrigée, session de révision IA, TodayPlan, seed, e2e et runbooks. La V1 documentée ici se concentre sur un objectif produit plus précis : transformer les QCM actuels en exercices fermés riches, typés, interactifs, sourcés et adaptés aux matières théoriques.

## Documents

- `ROADMAP_V1_RICH_CLOSED_QUESTIONS.md` : vision stratégique et architecture cible.
- `RICH_CLOSED_QUESTION_TYPES_CATALOG.md` : catalogue des types fermés, avec règles de validation.
- `RICH_CLOSED_QUESTIONS_DROIT_CONSTITUTIONNEL_EXAMPLES.md` : exemples courts et synthétiques de droit constitutionnel.
- `ROADMAP_EXECUTION_PLAN_V1.md` : lots atomiques V1.
- `ROADMAP_EXECUTION_LOT_V1_001_RICH_QUESTIONS_ROADMAP.md` : rapport du lot documentaire V1-001.

## Frontière avec la V0
