# Exemples — Questions riches fermées en droit constitutionnel

Ces exemples sont courts et synthétiques. Le fichier `Cours 2024-2025.pdf` n'était pas présent dans les emplacements inspectés pendant V1-001 ; aucune portion longue du cours source n'est copiée ici. Les thèmes reprennent des familles pédagogiques classiques : régimes politiques, État, Constitution, souveraineté, contrôle de constitutionnalité, institutions britanniques et américaines, IIIe/IVe République et modes de scrutin.

## `case_qualification` — 5 exemples

### 1. Responsabilité gouvernementale

- Objectif pédagogique : qualifier un régime à partir d'un critère institutionnel.
- Prompt utilisateur : “Un gouvernement doit conserver la confiance d'une chambre élue qui peut le renverser. Quelle qualification est la plus pertinente ?”
- Choix/options : régime parlementaire ; régime présidentiel ; confédération ; État régionalisé.
- Bonne réponse : régime parlementaire.
- Explication courte : la responsabilité politique du gouvernement devant le Parlement est un critère central du parlementarisme.
- Source conceptuelle : chapitre régimes parlementaire/présidentiel.
- Pourquoi plus riche qu'un QCM basique : l'étudiant applique un critère à un cas.

### 2. Séparation stricte

- Objectif pédagogique : distinguer séparation stricte et collaboration des pouvoirs.
- Prompt utilisateur : “L'exécutif et le législatif sont élus séparément et ne peuvent pas se renverser politiquement. Quelle qualification convient ?”
- Choix/options : régime présidentiel ; régime parlementaire ; régime d'assemblée ; monarchie parlementaire.
- Bonne réponse : régime présidentiel.
- Explication courte : la séparation stricte se traduit par une indépendance organique plus nette.
- Source conceptuelle : chapitre séparation des pouvoirs.
- Pourquoi plus riche qu'un QCM basique : le choix repose sur un faisceau d'indices.

### 3. État fédéral

- Objectif pédagogique : qualifier une forme d'État.
- Prompt utilisateur : “Des entités disposent d'une autonomie constitutionnellement garantie et participent à l'exercice du pouvoir fédéral. Quelle forme d'État est visée ?”
- Choix/options : État fédéral ; État unitaire déconcentré ; confédération ; État unitaire centralisé.
- Bonne réponse : État fédéral.
- Explication courte : l'autonomie garantie et la participation au pouvoir central caractérisent le fédéralisme.
- Source conceptuelle : chapitre formes de l'État.
- Pourquoi plus riche qu'un QCM basique : il faut combiner autonomie et participation.

### 4. Constitution matérielle

- Objectif pédagogique : distinguer constitution formelle et matérielle.
- Prompt utilisateur : “Un ensemble de règles organise les pouvoirs publics, même si certaines ne figurent pas dans un document intitulé Constitution. Quelle notion est en jeu ?”
- Choix/options : constitution matérielle ; constitution souple ; coutume électorale ; loi ordinaire.
- Bonne réponse : constitution matérielle.
- Explication courte : la constitution matérielle désigne les règles constitutionnelles par leur objet.
- Source conceptuelle : chapitre Constitution.
- Pourquoi plus riche qu'un QCM basique : le cas oblige à raisonner par fonction de la règle.

### 5. Référendum plébiscitaire

- Objectif pédagogique : distinguer référendum et plébiscite.
- Prompt utilisateur : “Une consultation populaire porte surtout sur la confiance accordée à un chef politique plutôt que sur le contenu d'une norme. Quelle qualification critique est pertinente ?”
- Choix/options : plébiscite ; contrôle a priori ; motion de censure ; bicaméralisme.
- Bonne réponse : plébiscite.
- Explication courte : le plébiscite personnalise la consultation autour d'un dirigeant.
- Source conceptuelle : chapitre démocratie directe et référendum.
- Pourquoi plus riche qu'un QCM basique : l'étudiant doit qualifier l'intention politique de la procédure.
